# SmartDoc Insight

SmartDoc Insight is an automated AI-powered knowledge base system designed to extract, categorize, and query information from unstructured documents (PDF, Docx) in real-time.

## Key Objectives

- **Automated Data Ingestion**: Automatically monitor folders for new documents and process them immediately using Python Watchdog.
- **AI-Powered Analysis**: Generate automatic summaries and classify documents into categories (Technical, Legal, Administrative, etc.) using LLMs like Google Gemini.
- **Semantic Search (RAG)**: Store document contents as vectors in ChromaDB to enable context-aware searching and natural language question answering.
- **Interactive Knowledge Base**: Provide a professional Streamlit-based dashboard for users to upload files and "chat" with their entire document library.
- **Efficient Retrieval**: Reduce information lookup time from minutes to seconds by retrieving relevant snippets directly from the source material.
